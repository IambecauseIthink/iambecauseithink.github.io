计算机基础知识

计算机的发展阶段

  • 第一代（1946～1958）：电子管数字计算机，奠定了计算机技术的基础

  • 第二代（1958～1964）：晶体管数字计算机

  • 第三代（1964～1971）：集成电路数字计算机

  • 第四代（1971～至今）：大规模和超大规模集成电路数字计算机

计算机基本结构及作用

计算机系统组成

  • 冯·诺伊曼提出“存储程序、程序控制”的设计思想

  • 计算机系统组成

    • 内部采用二进制的形式表示数据和程序

    • “存储程序”，即程序和数据一起存储在内存中，计算机按照程序顺序执行

    • 计算机由五个基本部分组成

      • 控制器

        • 计算机的控制中心，主要作用是智慧计算机的各部件能自动协调地工作

          • 包括

            • 寄存器

            • 译码器

            • 程序计数器（PC）

            • 操作控制器

          • 主要工作

            • 反复从存储器中逐条取出指令、分析指令和执行指令

      • 运算器

        • 是一个信息加工的部件，在控制器的指挥下，它对数据可以进行算术运算和逻辑运算

          • 包括

            • 算术逻辑运算单元（ALU）

            • 加法器

            • 通用寄存器

          • 主要功能

            • 实现对数据的算术运算和逻辑运算

            • 暂时存放参与运算的数据和某些中间结果

            • 挑选参加运算的数据，选中要执行的运算的功能，并把运算结果输出到指定部件中

      • 存储器

      • 输入设备

      • 输出设备

    • 两大部分

      • 硬件系统

      • 软件系统

计算机硬件系统

    • 输入设备

    • 输出设备

性能指标

  • 字长

    • 计算机内部一次可以处理的二进制位数，CPU的主要参数

    • 字长越长，可以表示的有效位数就越多，运算精度越高，计算机的处理能力越强

  • 主频

    • 用来表示CPU的运算速度

    • 主频越高，CPU的运算速度越快

    • 主频 = 外频 * 倍频系数

  • 运行速度

    • 每秒所能执行的指令数，单位位MIPS（百万条指令/秒）

  • 存取周期

    • 指对存储器进行连续存取操作所匀速的最短时间间隔

    • 存取周期越短，存取速度越快

  • 存储容量

    • 内存的容量，内存容量越大，所能存储的数据和运行的程序越多，程序的运行速度越快

信息与信息技术概述

信息的定义和表示

  • 信息就是通过语言、文字、图形/像、声视频等传播的内容。

    信息 不等于 消息
    信息是消息的内核，消息是信息的外壳

    信息 不等于 知识
    信息不是某种信号，知识是加工提炼后的信息

    数据（字母、数字符号、语音、图形/图像）是信息的载体

  • 普遍性

  • 时效性

  • 依附性（载体性）

  • 价值性

  • 共享性

  • 真伪性

数制与信息编码

数制的基本概念

会计算

  • 数制也称计数制，用一组固定的符号和统一的规则来表示数值的方法。通常的数制都采用进位计数制。

    • 数码

      • eg 十进制，数码为0～9

    • 基数

      • 数码的个数

        • eg 十进制，基数为10

    • 位权

      • 数码的权重

        • eg 十进制123.4，
          1 - 2；
          2 - 1；
          3 - 0；
          4 - -1

不同进制之间的转换

会计算

计算机中字符的编码

  • 西文字符的编码

    • ASCII码

      • 美国标准信息交换码

      • 由7位二进制数组成，可表示128种字符（实际占用8个二进制位，最高位补0）

      • 控制字符：前32个字符+空格+Del

      • 图形字符：可显示和打印（94个）

  • 汉字的编码

    • 汉字输入码

      • 数字编码

      • 字音编码

      • 字形编码

      • 音形编码

    • 国标码

      • 以国家标准局公布的GB2312-80规定的汉字交换码作为标准进行编码

        • 国标码（7445）

          • 汉字（6763个）

            • 一级汉字（3755个）

            • 二级汉字（3008个）

          • 字符符号（682个）

        • 全部国标码种的字符划分为94（（区）*94（位），每个字符由区号和位号来确定，即区位码。

    • 汉字内码

      • 是汉字在计算机内部存储、处理和传输用的信息代码

      • 将国标码的美国字节加80H

    • 汉字字形码

      • 点阵字形

        • 不论汉字的笔划多少，都可在同样大小的方块中写出，从而把方块分割成许多小方块，组成一个点阵，每个小方块就是二进制的一个位。

      • 矢量字形

        • 存储是描述汉字字形的轮廓特征，当要输出汉字时，通过计算机的计算，有汉字字型描述生成所需大小以及形状的汉字点阵。

    • 汉字地址码

      • 汉字字库中存储汉字字形信息的逻辑地址码。以简化内码到地址码的转换。

  • 二维码

    • 是用特定的集合图形按照一定规律在平面（二维方向）上分布的黑白相间的图形。

      • 优点

        • 高密度编码，信息容量大

        • 编码范围广

        • 容错能力强，具有纠错功能

        • 译码可靠性高

        • 可引入加密措施

        • 成本低，易制作，持久耐用

        • 条码符号形状，尺寸大小比例可变

        • 可使用激光或CCD阅读器识别

      • 缺点

        • 成为手机病毒、钓鱼网站传播的新渠道。

HTTP、HTTPS和FTP的区别

发表于 2021-03-03 更新于 2021-03-05 分类于 计算机科学与软件工程学习

面试的时候被问到了这个问题，在此记录一下

HTTP和FTP的区别

简单来说，HTTP（ Hyper Text Transfer Protocol ）是超文本传输协议，是面向网页的；FPT（File Transfer Protocol）是文件传输协议，是面向文件的。详细可以参见这篇

HTTP和HTTPS的区别

简单来说 HTTPS 比 HTTP 多了安全套接层 SSL（Secure Socket Layer）的加密，即 HTTPS 访问的浏览器与服务器之间收发的信息传输将更加安全可靠，但其代价就是 Web 服务器启用SSL需要获得一个服务器证书并将该证书与要使用SSL的服务器绑定，因此客户端在访问的时候必须进行密钥操作，降低了访问速度。详细可以参见这篇

科目一：综合素质（中学）

（本科目对应的思维导图附在文末）

教师职业理念

教育观

  • 素质教育的内涵

    • 面向全体学生

    • 促进学生全面发展

    • 促进学生个性发展

    • 以培养创新精神和实践能力为重点

  • 评价观

    • 评价目的是促进发展

    • 评价内容多元化

    • 评价方式多样化

    • 评价主体的多元化

    • 更注重评价过程

  • 区别于应试教育

    • 教育对象

      • 全体学生

    • 教育目的

      • 在教育活动中德智体美劳全面进行

    • 能力培养

      • 重视所有能力的培养

    • 学生评价

      • 发展性评价、评价方式多元化、评价主体多元化

    • 教学方法

      • 启发式、探究式教学

    • 教学内容

      • 降低教学内容难度、弱化学科体系

    • 教学着眼点

      • 重视发展性、终身教育、终身学习

学生观

  • 以人为本

    • 学生有独立意义的主体

      • 学生在教育活动中处于主体地位

      • 学生具有个体独立性，不以教师的意志为转移

      • 学生在教育活动中具有主体的需求与权责

    • 学生是发展的人

      • 学生的身心发展具有规律性

      • 学生具有巨大的发展潜能

    • 学生是独特的人（个性差异）

      • 以个性发展为基础

      • 因材施教

  • 现代学生观

    • 学生是一个完整的生命个体

    • 学生个体之间存在很大的差异性

    • 学生身上蕴藏着巨大的潜能

    • 学生的成长需要人文关怀

教师观

  • 现代教师观

    • 教师是学生学习的促进者

    • 教师应该是教育教学的研究者

    • 教师是课程的建设者和开发者

    • 教师应是社区型的开放教师

    • 教师应该是终身学习的践行者

  • 现代教学观

    • 教学是课程创生和开发的过程

    • 教学是师生交往、积极互动、共同发展的过程

    • 教学过程重于教学结果

    • 教学更关注人而不只是科学

  • 新课改下教师角色转换

    • 教师与学生的关系

      • 教师是学生学习的引导者，学生发展的促进者

    • 教学与课程的关系

      • 教师是课程的建设者和开发者

    • 教学与研究的关系

      • 教师是教育教学的研究者和反思的践行者

    • 学校与社区的关系

      • 教师是社区型开放教师

  • 新课改下教师行为转变

    • 对待师生关系

      • 尊重、赞赏

    • 对待教学

      • 帮助、引导

    • 对待自我

      • 反思

    • 对待与其他教育者的关系

      • 合作

教师职业道德规范

速记：三爱两人一终身

  • 爱国守法

    • 依法执教，不罚款

  • 爱岗敬业

    • 乐于奉献，不敷衍塞责

  • 关爱学生

    • 公平公正对待学生，尊重学生人格，不体罚或变相体罚学生

  • 教书育人

    • 因材施教，不以分数作为唯一评价标准

  • 为人师表

    • 廉洁从教，抵制有偿家教

  • 终身学习

    • 教学研究，勇于创新
